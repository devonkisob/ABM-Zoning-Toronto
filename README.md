# Toronto Missing Middle ABM

A census-tract-based agent-based simulation of housing affordability under alternative missing-middle zoning policies in Toronto.

## Team
- Devon Kisob
- Chanuth Weeraratna
- Kevin Kim

## Project structure
- `data/`: raw, interim, and processed datasets
- `src/`: reusable Python code for preprocessing, models, and simulation
- `notebooks/`: exploratory analysis and results notebooks
- `scripts/`: runnable scripts for preprocessing and simulation
- `results/`: figures and tables
- `docs/`: notes and milestone writing support
