# Data Sources

Raw data files are excluded from git due to file size. Download and place them in `data/raw/` before running the preprocessing scripts.

## 1. StatCan Census Profile 2021
**File:** `98-401-X2021007_English_CSV_data.csv` (~2.63 GB)  
**Catalogue:** 98-401-X2021007  
**URL:** https://www12.statcan.gc.ca/census-recensement/2021/dp-pd/prof/details/download-telecharger.cfm  
**Instructions:** Select "Census metropolitan areas (CMAs), tracted census agglomerations (CAs) and census tracts (CTs)" → CSV

## 2. StatCan Census Profile 2016
**File:** `98-401-X2016043_English_CSV_data.csv` (~1.5 GB)  
**Catalogue:** 98-401-X2016043  
**URL:** https://www12.statcan.gc.ca/census-recensement/2016/dp-pd/prof/details/download-telecharger.cfm  
**Instructions:** Same row as above, CSV format

## 3. StatCan CT Boundary Shapefile (2021)
**Folder:** `data/raw/ct_boundaries/lct_000b21a_e/`  
**Catalogue:** 92-160-X  
**URL:** https://www12.statcan.gc.ca/census-recensement/2021/geo/sip-pis/boundary-limites/index-eng.cfm  
**Instructions:** Select Census tracts → 2021 → Shapefile → unzip into `data/raw/ct_boundaries/`

## 4. TTC Routes and Schedules (GTFS)
**Folder:** `data/raw/ttc_gtfs/` (needs: `stops.txt`, `routes.txt`, `trips.txt`, `stop_times.txt`)  
**Source:** City of Toronto Open Data  
**URL:** https://open.toronto.ca/dataset/ttc-routes-and-schedules/  
**Licence:** Open Government Licence - Toronto  
**Instructions:** Download the GTFS zip and unzip into `data/raw/ttc_gtfs/`

## Processed Files (in git)
These are generated by the preprocessing scripts and committed to the repo so teammates don't need to re-run the full pipeline:

| File | Description |
|------|-------------|
| `data/processed/ct_agents_init.csv` | 1,220 Toronto CTs with 2021 census attributes |
| `data/processed/ct_calibration.csv` | 1,092 matched CTs with 2016-2021 delta_units and ML features |
| `data/processed/ct_transit_indicators.csv` | Transit proximity scores per CT (TTC GTFS) |
| `data/processed/stage1_model.pkl` | Fitted Stage 1 logistic regression |
| `data/processed/stage2_model.pkl` | Fitted Stage 2 linear regression |
| `data/processed/feature_scaler.pkl` | Fitted StandardScaler |
| `data/processed/results_s*.npy` | Simulation results for each scenario (N=100, T=40) |
